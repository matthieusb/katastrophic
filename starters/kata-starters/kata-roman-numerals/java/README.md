# kata roman numerals

Examples:

| Roman Symbol | Arabic number |
|--------------|---------------|
| I            | 1             |
| II           | 2             |
| V            | 5             |
| IX           | 9             |
| X            | 10            |
| XII          | 12            |
| XL           | 40            |
| L            | 50            |
| XC           | 90            |
| C            | 100           |
| CD           | 400           |
| D            | 500           |
| M            | 1000          |

## Story 1

Write a method to convert arabic numbers to roman numbers. For example:

```
1 --> I
10 --> X
7 --> VII
```

## Story 2

Write a method to convert roman numbers to arabic numbers. For example:

```
I --> 1
X --> 10
VII --> 7
```
